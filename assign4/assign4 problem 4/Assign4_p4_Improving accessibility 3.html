Semantic Elements for Better Meaning

HTML5 provides tags to clarify the content's purpose. This is better for both humans and screen readers:

  • <header>: Page or section introduction.
  • <nav>: Navigation links.
  • <main>: The primary page content.
  • <article>: Self-contained pieces (like this one).
  • <footer>: Bottom-of-page information.
